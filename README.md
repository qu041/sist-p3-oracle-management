# 쌍용교육센터 관리시스템 (3조 2차 프로젝트 - Oracle)

## 프로젝트 소개
이 프로젝트는 쌍용교육센터의 학사 관리 및 운영 효율화를 위해 설계된 데이터베이스 관리 시스템입니다. Oracle DB를 기반으로 구축되었으며, 요구사항 분석부터 ERD 설계, SQL 구현까지의 전 과정을 포함하고 있습니다.

## 프로젝트 산출물 구조

이 저장소는 프로젝트의 단계별 산출물을 다음과 같이 정리하고 있습니다:

### 1. 요구분석서
- 프로젝트의 목표와 기능적/비기능적 요구사항을 정의한 문서입니다.

### 2. 순서도
- 주요 업무 프로세스의 흐름을 시각화한 순서도입니다.

### 3. ERD (Entity Relationship Diagram)
- 데이터베이스의 논리적/물리적 모델링 설계도입니다.
![ERD](<3. ERD/3조 물리_논리 ERD.png>)

### 4. 테이블 정의서(DDL)
- 데이터베이스 테이블 생성, 수정, 삭제를 위한 DDL 스크립트 및 명세서입니다.

### 5. 데이터 정의서(DML)
- 초기 데이터 적재 및 테스트를 위한 DML 스크립트입니다.

### 6. (ANSI-SQL)쿼리문 정의서
- 주요 업무 로직 처리를 위한 표준 SQL 쿼리문 모음입니다.

### 7. (PL-SQL)오브젝트 스크립트
- 프로시저, 함수, 트리거 등 DB 내장 객체 스크립트입니다.

### 8. PPT
- 프로젝트 발표 및 최종 보고를 위한 프레젠테이션 자료입니다.

### 9. 최종 요약본
- 프로젝트의 핵심 내용과 결과를 요약한 문서입니다.
